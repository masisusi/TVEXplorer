# TV Show Search App

## Description
TV Show Search App is a simple web application that allows users to search for TV shows and save their favorites. The app fetches data from an external API and displays results dynamically.

#### Live demo URL: https://masisusi.github.io/TVEXplorer/

## Key Technical Features
- Dynamic rendering of search results using JavaScript
- Event handling with addEventListener (no inline handlers)
- localStorage for saving favorite shows
- Basic input handling and conditional rendering (e.g. "More" button)

## Local Setup

### Requirements
- A modern web browser (tested on Chrome, Edge)

### Run on Windows / macOS
1. Download or clone the repository  
2. Navigate to the project folder  
3. Open `index.html` in your browser

## Reflection

This project helped me practice working with APIs, DOM manipulation and handling user interactions. Compared to my previous project, I spent less time planning and structuring the application, which is also visible in the simplicity of the final result.

I did, however, improve my understanding of event handling by replacing inline onclick attributes with addEventListener, which makes the code cleaner and easier to maintain. I also got more comfortable using Git Bash for version control and pushing updates to GitHub.

One challenge was staying consistent with the workflow, especially since I was working on this project alongside other tasks. If I had more time, I would improve the user experience by adding better error handling and loading states.

Overall, the project strengthened my basic JavaScript skills and gave me more confidence in building small interactive web applications.
